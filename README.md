## What is RAG?

RAG stands for *Retrieval-Augmented Generation*.
In simple terms, it’s a way to make AI models smarter by letting them look up information from external data before answering a question.

Instead of relying only on what the model already knows, a RAG system first searches through documents (like PDFs, text files, or databases), finds the most relevant pieces of information, and then uses that context to generate a more accurate answer.

This makes responses more reliable, especially when working with custom or domain-specific data.
