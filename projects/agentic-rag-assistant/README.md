# Agentic AI Systems Portfolio

This repository showcases multiple **AI engineering projects** built using LangGraph, RAG, and LLM-based architectures.

Each project demonstrates real-world Agentic AI workflows, retrieval systems, and production-ready design thinking.

---

# Projects Overview

---

## 1. Agentic RAG AI Assistant

### What it does:
An intelligent AI system that answers user questions using uploaded documents and web data.

### Speciality:
- Uses LangGraph for multi-step reasoning
- Classifies queries into simple and complex
- Uses Retrieval-Augmented Generation (RAG)
- Integrates Pinecone vector database
- Generates structured AI responses

### Tech Used:
Python, LangGraph, Gemini, Pinecone, FastAPI

---

## 2. Multi-Document RAG System

### What it does:

### Speciality:
- Handles PDF, DOCX, TXT files
- Smart chunking and cleaning pipeline
- Semantic search using embeddings

### Tech Used:
Python, Gemini Embeddings, Pinecone
